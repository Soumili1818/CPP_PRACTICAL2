
Struct :-
1 By default, all the members of the structure are public.
2 The structure will automatically initialize its members.
3 When a structure is implemented, memory allocates on a stack.
4 Variables in a structure cannot be initialized during the declaration, but they can be done in a class.
5 There can be no null values in any structure member.
6 A structure is a value type.


Class :-
1 all members of the class are private.
2 constructors and destructors are used to initialize the class members.
3 memory is allocated on the heap in class.
4 the class variables may have null values.
5 a class is a reference type.
